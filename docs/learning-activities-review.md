# 듣기 퀴즈와 핵심 표현 생성 결과

오디오가이드 대본과 대표 장소 설명을 활용해 주요 어휘 10개에 대한 학습 활동을 생성했습니다.

## 생성 결과

- 대상 어휘: 10개
- 오디오 기반 활동: 5개
- 핵심 표현: 40개
- 퀴즈: 19개

## 생성 방식

- 오디오가이드 이야기가 있으면 해당 대본을 우선 사용합니다.
- 오디오가이드 이야기가 없으면 대표 장소의 `overview`를 사용합니다.
- 핵심 표현은 어휘가 직접 포함된 문장, 문화/역사/음식/체험 신호가 있는 문장을 우선 추출합니다.
- 퀴즈는 설명 기반 선택형과 어휘 빈칸형으로 생성합니다.

## 품질 메모

- `경복궁`, `한복`, `김치`, `국밥`, `갯벌`은 오디오 기반 퀴즈가 생성되었습니다.
- `남산 서울 타워`, `떡볶이`, `태권도`, `판소리`, `감귤`은 대본 또는 장소 설명 기반 활동이 포함됩니다.
- 현재 퀴즈는 자동 생성이므로, 실제 서비스 전에는 문항 자연스러움과 오답 선택지를 사람이 검수하는 단계가 필요합니다.
