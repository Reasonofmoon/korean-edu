# 주요 어휘 10개 대표 장소 검수

국문 관광정보 API 검색 결과를 바탕으로 학습 화면에 먼저 보여줄 대표 장소를 선정했습니다.

선정 기준:
- 어휘 자체와 제목/콘텐츠 주제가 직접 연결되는가
- 관광지/문화시설/행사/음식점 등 콘텐츠 타입이 학습 맥락에 맞는가
- 상세 설명과 이미지가 존재하는가
- 외국인 한국어 학습자가 실제 미션을 떠올리기 쉬운가

| 어휘 | 대표 장소 | 콘텐츠 타입 | 선정 방식 | 검수 메모 |
| --- | --- | --- | --- | --- |
| 경복궁 | 경복궁 | 관광지 | 수동 고정 | 동명의 음식점이 있어 공식 관광지를 대표로 고정 |
| 남산 서울 타워 | 남산서울타워 | 관광지 | 수동 고정 | 원 검색어는 0건이라 `남산서울타워` 별칭으로 매칭 |
| 한복 | 한국한복진흥원 | 문화시설 | 수동 고정 | 지역 창작소보다 한복 주제 대표성이 높아 수동 고정 |
| 김치 | 광주김치축제 | 축제/공연/행사 | 수동 고정 | 임의 김치찌개 식당보다 문화 학습 맥락이 강함 |
| 국밥 | 가야밀면돼지국밥 일산본점 | 음식점 | 자동 1위 | 음식점 후보가 많아 추후 지역/리뷰 기준 보강 필요 |
| 떡볶이 | 갈현동할머니떡볶이 | 음식점 | 자동 1위 | 서울 소재 실전 주문 미션에 적합 |
| 태권도 | 국기원(세계태권도본부) | 문화시설 | 수동 고정 | 태권도 대표 기관으로 학습 맥락이 명확 |
| 판소리 | 고창판소리박물관 | 문화시설 | 수동 고정 | 판소리 문화 설명과 방문 미션에 적합 |
| 갯벌 | 고창갯벌 | 관광지 | 수동 고정 | 자연/지역문화 미션에 적합 |
| 감귤 | 가을동화 감귤밭 | 관광지 | 자동 1위 | 제주 체험형 미션에 적합 |

## 다음 검수 포인트

- `국밥`, `떡볶이`, `감귤`은 현재 자동 1위입니다. 추후 서울/제주 중심 MVP 여부에 따라 수동 대표 장소를 바꿀 수 있습니다.
- 행사 콘텐츠는 기간성이 있으므로 운영 기간/상태 정보가 추가로 필요합니다.
- 음식점은 “추천/비추천” 표현보다 “주문 연습 가능 장소 후보”로 표시하는 것이 안전합니다.
