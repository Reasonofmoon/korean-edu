# 학습 상태 저장/복습/배지 구현 메모

프로토타입에 브라우저 로컬 저장 기반 학습 상태를 추가했습니다.

## 구현 내용

- 저장소: `localStorage`
- 저장 키: `kculture-learning-progress-v1`
- 저장 데이터:
  - 퀴즈 ID
  - 선택한 답
  - 정답 여부
  - 오답 횟수
  - 관련 어휘
  - 풀이 시각
  - 오디오 완료 여부
  - 오디오 완료 시각

## 화면 반영

- 상단 지표:
  - 정답률
  - 복습 필요 퀴즈 수
  - 완료 배지 수
  - 듣기 완료 수
- 어휘 카드:
  - 퀴즈 진행 상태
  - 모든 퀴즈 정답 시 완료 표시
- 퀴즈 카드:
  - 저장된 답안 복원
  - 정답/오답 색상 표시
  - 해설 표시
- 복습 카드:
  - 미풀이 또는 오답 퀴즈 노출
  - 오답 횟수가 많은 퀴즈를 우선 노출
  - 해당 어휘로 이동 버튼 제공
- 완료 배지:
  - 해당 어휘의 모든 퀴즈를 맞히고, 오디오가 있는 경우 오디오를 끝까지 들으면 배지 표시
- 학습 기록 초기화:
  - 확인창을 거친 뒤 모든 로컬 진도 삭제

## 다음 개선 포인트

- 정답률 추이를 날짜별로 저장
- 오답 횟수 기반 복습 우선순위 정렬
- 서버 로그인 기반 사용자 진도 저장
- 단어별 완료 조건을 퀴즈뿐 아니라 오디오 재생/표현 확인까지 확장
