# UX 넛지 적용 메모

확인일: 2026-05-13 KST

## 적용 내용

- 전체 데이터 섹션을 기본 접힘 구조로 전환
- 오늘의 추천 학습 행동 카드 추가
- 선택 어휘 상세에 `듣고 시작`, `퀴즈 풀기`, `장소 보기` CTA 추가
- 선택 어휘 상세를 단계형 접힘 패널로 정리
- 학습 활동과 복습 영역은 바로 행동할 수 있도록 기본 오픈 유지

## B=MAP 관점

- Behavior: 어휘 하나를 선택해 듣기/표현/퀴즈 중 하나를 완료
- Motivation: 완료 배지, 정답률, 복습 개수 표시
- Ability: 전체 데이터를 접고, 오늘 할 카드 한 장으로 시작점을 줄임
- Prompt: 추천 학습 카드와 상세 CTA 버튼

## app-factory CLI 확인

`app-factory` 명령은 설치되어 있으나 현재 로컬 설치가 깨져 실행되지 않았습니다.

```text
Cannot find module ... packages/cli/src/commands/list.js
```

이번 변경은 CLI 생성물 대신 기존 정적 프로토타입에 직접 적용했습니다.
