# 첫 학습 세션 플로우

기준일: 2026-05-13 KST

## 목적

app-factory의 `learning-content-app` / `builder-mini-behavior` 관점에서 첫 세션의 행동 루프를 명확히 한다.

첫 세션의 성공은 사용자가 전체 데이터를 탐색하는 것이 아니라, 어휘 하나를 골라  
`뜻 확인 → 듣기/읽기 → 표현 → 명대사 → K-pop 미션 → 퀴즈 → 완료/복습`까지 진행하는 것이다.

## Flow Map

1. 진입 — 오늘의 추천 카드 / 미완료 어휘 추천
2. 어휘 선택 — 카드 또는 CTA
3. 맥락 확인 — 뜻, 예문, 현장 미션, 로마자 병기 (자동 완료 간주)
4. 듣기/읽기 — 오디오 ended 또는 `읽기 완료했어요`
5. 핵심 표현 확인 — `표현 확인했어요`
6. **명대사 확인** — keywordHints 매칭 명대사 + `명대사 확인했어요` (완료 조건)
7. **K-pop 미션** — Melon 차트 트랙 표현 + `이 표현으로 말해 봤어요` (차트 있을 때)
8. 퀴즈 — 정답/오답 · 오답 횟수 저장
9. 완료/복습 — 배지, 복습 카드 **다시 풀기**, 어휘별/전체 진도 초기화

### 완료 조건

- 퀴즈 전체 정답
- 필요한 오디오 또는 읽기 완료
- 핵심 표현 확인 (표현이 있을 때)
- **명대사 확인**
- **K-pop 미션** (차트 데이터가 있을 때)

## State Machine

| State | 진입 조건 | 주요 UI | 다음 전이 |
| --- | --- | --- | --- |
| `pick` | 세션 시작, 선택 어휘 없음 | 어휘 카드, 추천 카드 | 어휘 선택 후 `context` |
| `context` | 어휘 선택됨 | 뜻, 예문, 현장 미션 | 자동으로 `listen` 또는 `expressions` |
| `listen` | 오디오 또는 읽기 대체 콘텐츠가 아직 완료 안 됨 | 듣기 카드, 오디오 플레이어, 읽기 완료 카드 | audio ended 또는 읽기 완료 클릭 후 `expressions` |
| `expressions` | 핵심 표현 미확인 | 표현 목록 | `quote` |
| `quote` | 명대사 미확인 | 명대사 카드, 확인 버튼 | `kpop` 또는 `quiz` |
| `kpop` | 차트 있고 K-pop 미완료 | 차트 미션 카드 | `quiz` |
| `quiz` | 미풀이/오답/다시 풀기 퀴즈 | 퀴즈 카드 | `complete` |
| `complete` | 완료 조건 충족 | 배지, 복습, 다음 어휘 CTA | 다음 어휘 |

## UI 반영

- `#session-flow`를 추가해 현재 선택 어휘의 단계 흐름을 시각화했다.
- 단계 버튼을 누르면 해당 섹션으로 이동한다.
- 선택 어휘 상세 상단 CTA가 현재 상태에 따라 바뀐다.
- 오디오가 없는 어휘는 읽기 완료 상태를 `localStorage`에 저장한다.
- 핵심 표현 확인 상태를 `localStorage`에 저장한다.
- 완료 조건에 핵심 표현 확인을 포함했다.
- 완료된 어휘에는 다음 미완료 어휘로 이동하는 CTA를 표시한다.
- 다음 어휘 CTA에는 `오답 복습 우선`, `같은 카테고리 이어 학습`, `다음 미완료 어휘` 이유를 표시한다.

## UX 개선 (2026-07-20)

- 복습 카드에서 **다시 풀기** (`pendingRetry`, 오답 횟수 유지)
- **이 어휘 진도만 초기화** / 전체 초기화 분리
- 완료 배지 클릭 시 해당 어휘로 이동

## UX 개선 (Quiz Focus)

- 퀴즈 단계는 전체 화면 **집중 모드** (`#quiz-focus`)로 진입한다.
- 한 문제씩 표시 · 진행 바 · 이전/다음 · 안 푼 문제로 이동 · ESC 닫기.
- 세션 CTA / 플로우 단계 / 복습 「다시 풀기」 / 상세 「집중 모드 퀴즈」에서 진입.
- 전 문항 정답 시 완료 화면 + 다음 어휘 CTA.

## UX 개선 (Culture Lexeme P2)

- 세션 단계 `culture` (`#culture-step`) — 명대사 **직전** 슬롯.
- 기본: **보너스** (완료 게이트 아님). K-drama / K-movie 관심사 또는 관련 여행 목적이면 **필수**.
- `progress.cultureLexemes[word]` 롤업 + `progress.cultureLexemes[seed-cl-*]` 개별 확인.
- 데이터: `seed-cl-*` 교육 시드만 (원문 대사 아님).
