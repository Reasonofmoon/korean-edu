# 관광지 오디오가이드 매칭 검수

`storySearchList`를 사용해 주요 어휘 10개와 오디오가이드 이야기를 매칭했습니다.

## 결과 요약

- 대상 어휘: 10개
- 관련 이야기 매칭: 7개 어휘
- 이야기 카드: 16개
- 오디오 URL 포함: 10개
- 오류: 0개

## 매칭 품질 메모

| 어휘 | 상태 | 메모 |
| --- | --- | --- |
| 경복궁 | 좋음 | 경복궁, 근정전 등 직접 관련 오디오가 매칭됨 |
| 남산 서울 타워 | 없음 | 넓은 `타워` 검색은 엉뚱한 지역 타워가 잡혀 제외 |
| 한복 | 보통 | 한복 거리/상가/공방 이야기가 매칭됨. 일부 오디오는 대본만 있음 |
| 김치 | 보통 | 돌산갓김치거리 오디오가 매칭됨 |
| 국밥 | 좋음 | 돼지국밥 관련 오디오가 매칭됨 |
| 떡볶이 | 없음 | 시장 일반 이야기는 제외. 직접 떡볶이 이야기는 없음 |
| 태권도 | 없음 | `태권` 검색의 엉뚱한 결과를 제외 |
| 판소리 | 보통 | 판소리박물관/전시관 대본이 매칭됨. 오디오 URL은 없음 |
| 갯벌 | 좋음 | 갯벌/생태 관련 오디오가 매칭됨 |
| 감귤 | 보통 | 제주감귤박물관 대본이 매칭됨. 오디오 URL은 없음 |

## 설계 판단

엉뚱한 오디오보다 빈 상태가 낫기 때문에, `남산 서울 타워`, `떡볶이`, `태권도`는 현재 오디오 이야기 없음으로 표시합니다.

프로토타입에서는 오디오 URL이 있으면 재생 카드로, 없으면 대본 기반 읽기/듣기 준비 카드로 표시합니다.
