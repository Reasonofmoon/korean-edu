# K-Culture Korean Learning MVP 설계

## 1. 제품 방향

이 MVP는 세종한국문화1/2 주요 어휘를 출발점으로 삼아, 한국관광공사 공공데이터를 연결하는 한국어 학습 서비스입니다. 목표는 단어 암기 앱이 아니라, 외국인 학습자가 실제 한국 문화 장소와 상황에서 한국어를 써 볼 수 있게 만드는 것입니다.

제품 한 줄 설명:

> 한국문화 어휘를 배우고, 그 어휘가 실제 한국 어디에서 어떻게 쓰이는지 미션으로 경험하는 학습 지도.

## 2. 핵심 사용자

- 해외에서 한국어와 K-culture를 배우는 초급-중급 학습자
- 한국 여행을 준비하는 외국인
- 한국에 거주하며 실생활 한국어 연습 장소가 필요한 외국인
- 세종학당/한국문화 수업을 보조할 디지털 자료가 필요한 강사

## 3. MVP 핵심 기능

### 3.1 문화 어휘 카드

CSV의 `주요 어휘`를 카드로 보여줍니다.

카드 필드:
- 한국어 어휘
- 교재명
- 단원명
- 관련 페이지
- 주제 카테고리
- 쉬운 한국어 설명
- 실전 예문
- 추천 미션

### 3.2 어휘 기반 장소/콘텐츠 매칭

어휘를 한국관광공사 API 키워드 검색에 연결합니다.

예시:
- `경복궁` -> 관광지 상세, 이미지, 오디오 가이드
- `한복` -> 전통문화시설, 궁궐 주변 한복 체험
- `김치`, `국밥`, `떡볶이` -> 음식점/시장/음식문화 콘텐츠
- `갯벌`, `감귤` -> 지역 자연/체험 콘텐츠
- `판소리`, `강강술래` -> 문화시설/행사/스토리 콘텐츠

### 3.3 로컬 미션

학습자가 실제 상황에서 쓸 수 있는 짧은 행동 과제를 제공합니다.

미션 예시:
- 시장에서 떡볶이 주문하기
- 궁궐에서 사진 부탁하기
- 식당에서 맵기 조절 요청하기
- 전시/공연에서 감상 표현 말하기
- 걷기 코스에서 날씨와 풍경 표현하기

### 3.4 Local Signal

공공데이터와 추후 로컬 멘토 데이터를 합쳐 장소별 학습 친화도를 표시합니다.

초기 MVP 점수 요소:
- 공식 관광정보 존재 여부
- 이미지/상세정보 존재 여부
- 관련 문화 어휘 수
- 오디오/스토리 콘텐츠 존재 여부
- 지역 방문 트렌드 참고 가능 여부
- 한국어 미션 생성 가능 여부

추후 확장 점수 요소:
- 초급 한국어 사용 난이도
- 혼밥/혼자 방문 가능성
- 외국인 접근성
- 가격/메뉴 투명성
- 로컬 멘토 검증
- 관광객 과밀도

## 5. API별 역할

| 데이터 | MVP 역할 | 우선순위 |
| --- | --- | --- |
| 세종한국문화 CSV | 어휘 커리큘럼, 단원/페이지 메타데이터 | 필수 |
| 국문 관광정보 서비스 | 장소/음식점/문화시설/행사/이미지 검색 | 필수 |
| 관광지 오디오 가이드 | 듣기 학습, 문화 이야기, 스토리 카드 | 높음 |
| 관광빅데이터 | 지역 방문 트렌드, 혼잡도 참고 지표 | 중간 |
| 두루누비 | 걷기 코스, 자연/지역문화 미션 | 중간 |

우선 어휘 10개:

- 떡볶이
